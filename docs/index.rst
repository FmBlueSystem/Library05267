Nueva Biblioteca
===============

Sistema de biblioteca musical con playlists inteligentes y análisis avanzado de audio.

.. toctree::
   :maxdepth: 2
   :caption: Contenido:

   user/index
   development/index
   api/index

Características Principales
-------------------------

* Gestión de biblioteca musical con soporte para múltiples formatos
* Playlists inteligentes basadas en reglas y metadatos
* Análisis de audio (BPM, tonalidad, energía, etc.)
* Sistema de recomendaciones basado en similitud musical
* Interfaz Material Design 3
* Exportación a múltiples formatos
* Optimizado para HiDPI

Contribuir
---------

Para contribuir al proyecto:

1. Fork el repositorio
2. Crea una rama para tu feature (`git checkout -b feature/nueva-caracteristica`)
3. Haz commit de tus cambios (`git commit -am 'Agregar nueva característica'`)
4. Push a tu rama (`git push origin feature/nueva-caracteristica`)
5. Crea un Pull Request

Para más detalles, consulta la :doc:`guía de contribución </development/contributing>`.
